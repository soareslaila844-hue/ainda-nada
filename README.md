=== APP TECH PAIN ===

COMO USAR:

1. MODO SIMPLES (Recomendado para testes):
   - Abra o arquivo "index.html" no navegador do celular
   - Funciona imediatamente

2. PARA COMPARTILHAR:
   - Envie a pasta completa "TechPainApp" para outras pessoas
   - Ou envie apenas o arquivo "index.html"

3. COMO INSTALAR COMO APP:
   Para Android:
   - Use o aplicativo "PWA Builder" da Play Store
   - Ou o "Web2Apk" para criar APK

   Para iPhone:
   - Abra no Safari
   - Toque em "Compartilhar"
   - Toque em "Adicionar à Tela de Início"

4. CONEXÃO COM ESP32:
   - Ative o Bluetooth no celular
   - Nome do dispositivo: "TechDevice"
   - Conecte pelo botão "Conectar Bluetooth"
